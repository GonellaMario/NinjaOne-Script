# NinjaOne-Script
NinjaOne PowerShell Scripts

This repository contains the publication of PowerShell scripts developed for NinjaOne.

These scripts are designed to be used within the NinjaOne platform for tasks such as:

IT automation

Device management and maintenance

Monitoring and remediation

Configuration deployment

Support and troubleshooting activities

Purpose

To provide modular, reusable, and easily deployable tools within the NinjaOne console, with a focus on:

Compatibility with Windows environments

Execution in SYSTEM context when required

Clear and structured logging

Proper error handling and security considerations

Requirements

NinjaOne RMM

PowerShell 5.1 or later (unless otherwise specified in the script documentation)

Each script includes a functional description and, where necessary, implementation notes for NinjaOne.
